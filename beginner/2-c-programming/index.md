# Beginner 2: C Programming Basics

## Learning Goals

- Learn C syntax, types, operators, and control flow.
- Write and debug small procedural programs.
- Build habits for memory-safe and readable C code.

## Topics

- C syntax and structure
- Functions and modular code
- Arrays, pointers, and basic memory concepts
