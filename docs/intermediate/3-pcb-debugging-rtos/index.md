# Intermediate 3: PCB, Debugging, and RTOS

## Learning Goals

- Design practical hardware using PCB workflows.
- Debug firmware with professional tooling.
- Understand RTOS scheduling, tasks, and concurrency.

## Topics

- Hardware and PCB design
- GDB and OpenOCD debugging
- RTOS fundamentals
